## 📊 功能模块

### 1. 参数输入模块
| 功能 | 说明 |
|------|------|
| 省份选择 | 31省机制电价自动加载 |
| 项目类型 | 光伏/风电/储能 |
| 存量/增量 | 136号文分类处理 |
| 装机规模 | MW级输入 |
| 资源参数 | 利用小时、衰减率 |
| 电价参数 | 机制/市场化比例 |
| 融资结构 | 资本金/贷款比例 |

### 2. 财务计算模块
| 指标 | 计算方法 | 投决标准 |
|------|---------|---------|
| 税前全投资IRR | Newton-Raphson迭代 | ≥6.5% |
| 税后资本金IRR | 考虑融资结构 | ≥8% |
| NPV | 现金流折现 | >0 |
| LCOE | 全生命周期成本/发电量 | <当地电价 |
| 动态回收期 | 折现现金流累计 | ≤10年 |

### 3. 敏感性分析模块
- **龙卷风图**: 5因素敏感度排序
- **情景分析**: 乐观/基准/悲观
- **盈亏平衡**: 临界值计算

### 4. 报告生成模块
- 投决会报告摘要
- 现金流明细表
- CSV导出

## 📚 136号文政策要点

### 存量项目(2025年6月1日前)
- 机制电价: 不高于当地煤电基准价
- 执行期限: 按现行政策
- 电量规模: 衔接现行保障电量

### 增量项目(2025年6月1日起)
- 机制电价: 竞价形成
- 竞价上限: 考虑合理成本收益
- 执行期限: 按同类项目回收期确定
