# ⚡ 新能源投资收益计算器

基于2025年136号文机制电价政策的新能源项目投资决策辅助工具。

## 功能特性

### 📊 核心财务指标
- **税前全投资IRR** - 项目本身盈利能力(投决会主要指标)
- **税后资本金IRR** - 股东实际回报率
- **净现值(NPV)** - 项目创造绝对价值
- **平准化度电成本(LCOE)** - 发电成本竞争力
- **动态/静态投资回收期** - 资金回收速度

### 🗺️ 政策数据库
内置全国31个省份机制电价政策:
- 山东: 0.3949元/kWh (首个落地省份)
- 广东: 0.453元/kWh (全国最高)
- 贵州: 0.3515元/kWh
- 新疆: 0.262元/kWh (西部最低)
- ...等等

### 📈 敏感性分析
- 龙卷风图 - 直观展示各因素敏感度排序
- 情景分析 - 乐观/基准/悲观三情景对比
- 盈亏平衡分析 - 各参数临界值

### 💾 数据导出
- 投决会报告生成
- 现金流明细表导出

## 投决会门槛标准

| 项目类型 | IRR门槛 | 回收期上限 |
|---------|--------|-----------|
| 集中式光伏 | ≥6.5% | 10年 |
| 分布式光伏 | ≥7.0% | 9年 |
| 陆上风电 | ≥6.5% | 10年 |
| 海上风电 | ≥6.0% | 12年 |

## 使用说明

### 1. 参数输入
- 选择省份(自动加载当地机制电价)
- 选择项目类型(光伏/风电/储能)
- 输入装机规模、利用小时
- 设置电价参数(机制/市场化比例)
- 配置融资结构

### 2. 查看结果
点击"计算收益"后，可查看:
- 财务指标总览
- 敏感性龙卷风图
- 年度现金流明细

### 3. 投决建议
系统根据大唐集团等发电集团标准自动生成:
- ✅ 通过/⚠️ 边缘/❌ 否决建议
- 风险提示
- 优化方向

## 136号文核心要点

### 存量项目(2025年6月1日前投产)
- 机制电价: 按现行政策，不高于煤电基准价
- 电量规模: 衔接现行保障政策
- 执行期限: 按现行政策期限

### 增量项目(2025年6月1日起投产)
- 机制电价: 竞价形成，按入选最高报价
- 电量规模: 根据消纳责任权重确定
- 执行期限: 按同类项目回收期确定
